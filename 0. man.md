자주 사용하는 Linux 명령어들
===========================
```
ls : 현재 디렉터리에 있는 파일 및 폴더 목록 나열
cd : 디렉터리 변경
mkdir : 디렉터리 생성
rm : 파일 삭제
rmdir : 빈 디렉터리 삭제
cp : 파일이나 디렉터리 복사
mv : 파일이나 디렉터리를 이동 혹은 이름을 변경
cat : 파일 내용을 출력
head : 파일 첫 부분을 출력
tail : 파일 끝 부분을 출력
touch : 빈 파일 생성, 파일의 수정 시간 변경
chmod : 파일이나 디렉터리 권한 변경
- chmod 파일디렉터리명 번호
chown : 파일이나 디렉터리 소유자 변경
-chown 사용자:그룹명 파일디렉터리명
ps : 프로세스 목록 출력
top : 시스템 프로세스 및 자원 사용량 모니터링
```

```
ps -ef | grep [찾을프로세스]

du -sh : 현재 디렉터리 사용량

du -sh ./* : 디렉터리들의 용량

df -hT 디스크 용량 확인

작업취소 Ctrl + c

clear 화면 클리어 (윈도우 cmd는 cls)
```
TOP
----
```
shift + p : CPU 사용률 내림차순
shit + m : 메모리 사용률 내림차순
shift + t : 프로세스가 돌아가고 있는 시간 순
k : kill. k 입력 후 PID 번호 작성. signal은 9
f : sort field 선택 화면 -> q 누르면 RES순으로 정렬
a : 메모리 사용량에 따라 정렬
b : Batch 모드로 작동
1 : CPU Core별로 사용량 보여줌
		
SHR 옆에 있는 S 항목으로 볼 수 있음
D : Uninterruptiable sleep. 디스크 혹은 네트워크 I/O를 대기
R : 실행 중(CPU 자원을 소모)
S : Sleeping 상태, 요청한 리소스를 즉시 사용 가능
T : Traced or Stopped. 보통의 시스템에서 자주 볼 수 없는 상태
Z : zombie. 부모 프로세스가 죽은 자식 프로세스

> 다음 쉘 명령으로 웹서버가 얼마나 많은 메모리를 사용하고 있는지 확인할 수 있다
ps -ylC httpd | awk '{x += $8;y += 1} END {print "Memory Usage (MB): "x/1024; print "Average Proccess Size (MB): "x/((y-1)*1024)}'
```

vi
```
i : 현재 위치에서 편집모드로 전환
esc : 명령 모드 전환
:w : 파일을 저장
:q : vi를 종료
:wq : 파일을 저장하고 vi 종료
:q! : 저장하지 않고 강제 종료
:set nu : 행 번호 표시
:gg : 맨 처음 행 이동
:G : 마지막 행 이동
/word : "word" 라는 단어를 검색
n : 다음 검색 결과 이동
N : 이전 검색 결과 이동
dd : 현재 행을 삭제
yy : 현재 행을 복사
p : 현재행을 다음행에 붙여넣기
```
